# Zipnova Seller Portal

Desarrollo frontend para una plataforma logística y de gestión e-commerce utilizada por sellers en Argentina.

Este proyecto forma parte de un entorno productivo donde los sellers se conectan al ecosistema de Zipnova mediante integraciones API para gestionar distintos procesos operativos y comerciales.

---

## Descripción

Zipnova Seller Portal permite a los sellers administrar diferentes aspectos de su operación desde una plataforma centralizada.

---

## Tecnologías utilizadas

- React
- JavaScript
- REST APIs
- Bootstrap / componentes UI

---

## Funcionalidades desarrolladas

### Integración con APIs
- Conexión de sellers mediante consumo de APIs
- Sincronización con backend
- Manejo de datos asincrónicos

---

## Contexto productivo

Este repositorio representa parte de una plataforma real utilizada dentro de un ecosistema logístico y e-commerce en Argentina.

El repositorio no contiene credenciales sensibles ni información privada de operación.
